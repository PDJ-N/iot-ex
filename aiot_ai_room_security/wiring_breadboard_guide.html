PIR 센서 + 3핀 부저 브레드보드 배선도

Raspberry Pi 5의 물리 1번 3.3V를 PIR 센서와 3핀 부저가 함께 써야 하므로, 브레드보드의 빨간 + 라인으로 전원을 나눠주는 방식이다.

1. 그대로 따라 꽂는 회로도

빨강: 3.3V 전원 검정: GND 노랑: PIR 신호 GPIO16 보라: 부저 신호 GPIO18

2. 왜 브레드보드가 필요한가?

PIR 센서의 V와 부저의 +가 둘 다 3.3V를 사용한다. 라즈베리파이 물리 1번 핀 하나에 선 두 개를 억지로 꽂지 말고, 브레드보드의 + 라인으로 전원을 나눠준다.

분배 라인 연결할 것
브레드보드 + 라인 Pi 3.3V, PIR V, 부저 +
브레드보드 - 라인 Pi GND, PIR G, 부저 -

3. 최종 연결표

부품 핀 꽂는 곳
Raspberry Pi 물리 1번 3.3V 브레드보드 + 라인
Raspberry Pi GND 아무 핀 브레드보드 - 라인
PIR 센서 S GPIO16, 물리 36번
PIR 센서 V 브레드보드 + 라인
PIR 센서 G 브레드보드 - 라인
3핀 부저 S GPIO18, 물리 12번
3핀 부저 + 브레드보드 + 라인
3핀 부저 - 브레드보드 - 라인

4. 꽂는 순서

  1. 라즈베리파이 전원을 끈다.
  2. Pi 물리 1번 3.3V를 브레드보드 빨간 + 라인에 연결한다.
  3. Pi의 GND 핀 하나를 브레드보드 파란/검정 - 라인에 연결한다.
  4. PIR V를 브레드보드 + 라인에 꽂는다.
  5. PIR G를 브레드보드 - 라인에 꽂는다.
  6. PIR S를 Pi 물리 36번, 즉 GPIO16에 꽂는다.
  7. 부저 +를 브레드보드 + 라인에 꽂는다.
  8. 부저 -를 브레드보드 - 라인에 꽂는다.
  9. 부저 S를 Pi 물리 12번, 즉 GPIO18에 꽂는다.
  10. USB 웹캠을 라즈베리파이 USB 포트에 꽂는다.
부저의 +를 GPIO18에 꽂지 않는다. 3핀 부저에서 GPIO18에 연결하는 핀은 반드시 S이다.
